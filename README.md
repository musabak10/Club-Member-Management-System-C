# 🎓 Club Member Management System (C)

A console-based **Club Member Management System** developed in **C**. The project demonstrates linked list implementation, dynamic memory allocation, recursion, and function pointers while managing university club members and their activity points.

---

## 🚀 Features

- Add new club members
- List all members
- Search members by student number
- Count members
- Calculate activity points
- Standard point calculation
- Bonus point calculation
- Dynamic memory management

---

## 🛠 Technologies Used

- C Programming Language
- Linked Lists
- Dynamic Memory Allocation (`malloc`, `free`)
- Function Pointers
- Recursion
- Structures
- Modular Programming
